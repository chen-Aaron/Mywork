MacOS

Mac OS下项目调试

开发环境

    在Mac下，开发调试流程如下:     从内网git仓库拷下项目，安装项目相关依赖即可。这里以Mac os Sierra v10.12.4为例:     首先检查现有node版本

node -v

    本项目采用目前较新版本v6.10.3。目前建议也采用该版本，node中文官网下下载安装6.10.3版本。或者通过Homebrew安装。

brew install node
//切换到指定版本
sudo n 6.10.3

    v6.10.3版本node集成了npm。npm的版本没有严格的要求所有，这里不做详细说明。不过由于国内网络问题下载国外资源速度会较慢，这里建议使用淘宝镜像来解决网络问题。

$ npm install -g cnpm --registry=https://registry.npm.taobao.org

淘宝镜像是由淘宝提供的国内npm解决方案。从 registry.npm.taobao.org 安装所有模块. 当安装的时候发现安装的模块还没有同步过来, 淘宝 NPM 会自动在后台进行同步, 并且会让你从官方 NPM registry.npmjs.org 进行安装. 下次你再安装这个模块的时候, 就会直接从 淘宝 NPM 安装了.

cd XXX
cnpm install 安装项目依赖
//完成之后，
cnpm run dev

    开发环境就基本搭建完成.
